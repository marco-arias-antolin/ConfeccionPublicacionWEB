ENUNCIADO

En una actividad en equipo anterior ha instalado la herramienta de edición web Visual Studio Code. Ya la conoce un poco y sabe de su utilidad y posibilidades.

En esta ocasión, le pedimos que instale las siguientes extensiones:

  1. Live preview
  2. Image preview
  3. Auto close tag
  4. 2077 theme

Y localice la información referente a dichas extensiones y explique en qué consisten. Remita, en un documento, a su tutor/a, cómo procede a la instalación de estas extensiones y su utilidad.

  1. Live preview: Esta extensión te permite previsualizar en tiempo real el resultado de tus cambios en un archivo HTML, CSS o Markdown. Al hacer una modificación en el código, la vista previa se actualiza automáticamente en la misma ventana de Visual Studio Code. Es útil para ahorrar tiempo al no tener que ir y venir entre el código y la vista previa en un navegador.
  2. Image preview: Esta extensión te permite previsualizar imágenes en Visual Studio Code sin tener que abrir una aplicación externa. Solo necesitas hacer clic derecho en la imagen y seleccionar "Image Preview" para verla en la ventana de Visual Studio Code. Es útil para ahorrar tiempo al no tener que cambiar constantemente entre la aplicación de codificación y una aplicación de visualización de imágenes.
  3. Auto close tag: Esta extensión es útil para cerrar automáticamente las etiquetas HTML, XML y JSX. Cuando escribes el símbolo "<", la extensión lo reconocerá y automáticamente cerrará la etiqueta correspondiente después de escribir el nombre de la etiqueta. Esto puede ahorrarte tiempo y reducir la probabilidad de errores de sintaxis.
  4. 2077 theme: Esta es una extensión de tema de Visual Studio Code, que te permite cambiar la apariencia visual de tu editor. El tema 2077 es un tema oscuro con una combinación de colores morados y azules. Es útil para reducir la fatiga visual y mejorar la legibilidad del código.

Para instalar estas extensiones en Visual Studio Code, sigue estos pasos:

  1. Abre Visual Studio Code y ve a la sección "Extensiones" en la barra lateral izquierda.
  2. Busca la extensión que deseas instalar (por ejemplo, "Live preview") y haz clic en "Instalar".
  3. Si la extensión requiere permisos adicionales, se te pedirá que los concedas antes de continuar con la instalación.
  4. Una vez que la extensión se haya instalado correctamente, aparecerá en la lista de extensiones instaladas en la sección "Extensiones".
  5. La instalación de estas extensiones puede mejorar tu flujo de trabajo y hacer que la codificación sea más eficiente.
