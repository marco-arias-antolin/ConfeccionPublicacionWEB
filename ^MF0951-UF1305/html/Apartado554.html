Identificación de los objetos predefinidos para manejar nuevas estructuras

Enunciado

Crear un botón en una página web es fundamental para un buen diseño pues muchas de las llamadas a la acción van incrustadas en un botón.

Por tanto, le solicitamos:

  • Cree un documento que contenga un botón y que al pulsarlo lleve a un periódico.
  • Tras pulsar el botón deberá abrirse una ventana donde el usuario otorgue su consentimiento para ingresar al periódico.

Actividad asociada al CE2.2

Identificar los objetos predefinidos por el lenguaje de guion para manejar nuevas estructuras y utilidades que añadirán nuevas funcionalidades a las páginas, de acuerdo a las especificaciones técnicas del lenguaje.

Solución

Actividad asociada al CE2.2

Conceptos y objetos del lenguaje de scripting que se utilizan en el código:

  1. window: Es un objeto global en JavaScript que representa la ventana del navegador. Proporciona métodos y propiedades para interactuar con la ventana y controlar su comportamiento.
  2. document: Es un objeto que representa el documento HTML cargado en la ventana del navegador. Proporciona métodos y propiedades para manipular el contenido y la estructura del documento.
  3. confirm: Es una función incorporada de JavaScript que muestra una ventana emergente con un mensaje y botones de confirmación ("Aceptar" y "Cancelar"). Devuelve true si el usuario hace clic en "Aceptar" y false si hace clic en "Cancelar".
  4. window.open: Es un método de la ventana del navegador que abre una nueva ventana o pestaña y carga una URL específica en ella. Toma varios parámetros opcionales, incluyendo la URL a cargar, el nombre de la ventana o pestaña, opciones de personalización y más.

Estos objetos y funciones son parte del lenguaje de scripting JavaScript y se utilizan para agregar funcionalidad dinámica a las páginas web. En el ejemplo dado, se utilizan para mostrar una ventana emergente de confirmación y redirigir al usuario al periódico si se otorga el consentimiento.
