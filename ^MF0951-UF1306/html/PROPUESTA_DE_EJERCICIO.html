EJERCICIO COMPLEMENTARIOS Y DE SUSTITUCIÓN

Una sala de proyección de cine nos encarga un proyecto para poder dar mantenimiento a las películas que proyecta y para ello se debe de desarrollar la siguiente herramienta Web.

En primer lugar se deberá de crear la Base de Datos BDCINE con la tabla películas que tiene la siguiente estructura:

Nombre de Campo
Tipo Tamaño Observaciones
Identificador VARCHAR 20 Campo Clave Primaria. Las películas tienen un identificador único que proporciona la distribuidora
Titulo VARCHAR 100 Campo índice
Genero VARCHAR 100 Género de la película: Ciencia ficción, aventuras, etc.
FechaInicio DATE Fecha en que se inicia la proyección
FechaFin DATE Fecha en que finaliza la proyección

Nota: Los nombres de los campos no lleva tildes.

Desde la página index.php y a través de un menú se accederán a las siguientes operaciones:

  • Alta de películas en la Base de Datos teniendo cuidado que no se repitan los identificadores.
  • Consulta de películas:
    • Lista General de películas
    • Lista de películas por título
    • Lista de películas por género
    • Lista de películas entre dos fechas de inicio
  • Las operaciones de consulta permitirán la modificación y eliminación de películas.

Este ejercicio tiene la posibilidad como proyecto de sustituir al resto de ejercicios que queden en ambas UF.
