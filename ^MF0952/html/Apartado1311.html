Clasificación de los comandos que realizan una modificación o generación en la estructura de almacenamiento

Enunciado

Cree una carpeta llamada ejemplo en el escritorio, dentro de la misma cree otra carpeta llamada ejemplo2, dentro de ella cree un archivo .txt. Utilice los comandos adecuados para que se muestre en el terminal de Windows todos los archivos contenidos en la carpeta ejemplo y sus subdirectorios.

Actividad asociada al CE1.2

Clasificar los mandatos y comandos de uso posible, para realizar la generación o modificación de la estructura de almacenamiento en el sitio web de acuerdo a unas instrucciones recibidas.

Solución

Abre el menú Inicio y busca "cmd" para abrir el símbolo del sistema (terminal de Windows).

Para crear la carpeta "ejemplo" en el escritorio:

mkdir C:\Users\TuUsuario\Desktop\ejemplo

Cambiar al directorio "ejemplo":

cd C:\Users\TuUsuario\Desktop\ejemplo

Crear la carpeta "ejemplo2" dentro de la carpeta "ejemplo":

mkdir ejemplo2

Cambiar al directorio "ejemplo2":

cd ejemplo2

Crear un archivo de texto llamado "archivo.txt":

echo Este es un archivo de ejemplo > archivo.txt

Cambia el directorio actual al escritorio utilizando el comando cd (si no estás en el escritorio):

cd C:\Users\TuUsuario\Desktop

Utiliza el comando dir con la opción /S para mostrar todos los archivos en la carpeta "ejemplo" y sus subdirectorios:

dir ejemplo /S

Actividad asociada al CE1.2

Creación de Carpetas y Archivos:

Crear una carpeta en el directorio raíz del sitio web:

mkdir nombre_carpeta

Navegar a la carpeta recién creada:

cd nombre_carpeta

Crear una subcarpeta dentro de la carpeta actual:

mkdir subcarpeta

Crear un archivo HTML en la carpeta actual:

echo "Título
Contenido
" > archivo.html

Crear un archivo CSS en la carpeta actual:

echo "selector { propiedad: valor; }" > estilo.css

Edición de Archivos:

Abrir un archivo HTML existente para edición (usando un editor de texto):

notepad archivo.html

Editar un archivo CSS existente (usando un editor de texto):

notepad estilo.css

Visualización de Contenido:

Listar todos los archivos y carpetas en el directorio actual:

dir

Mostrar el contenido de un archivo en la consola:

type archivo.html

Navegar a una carpeta específica:

cd ruta_absoluta_o_relativa

Otros:

Copiar un archivo de una carpeta a otra:

copy origen\archivo.html destino\

Mover una carpeta o archivo a otra ubicación:

move origen\subcarpeta destino\

Eliminar un archivo:

del archivo.html

Eliminar una carpeta y su contenido:

rmdir /s /q nombre_carpeta

Estos comandos pueden variar según el sistema operativo (por ejemplo, Windows o Linux) y la interfaz de línea de comandos que estés utilizando. Adicionalmente, hay que tener en cuenta que trabajar en un sitio web en producción puede requerir más consideraciones, como asegurarte de tener copias de seguridad y ser consciente de cómo los cambios afectan a la experiencia del usuario.
