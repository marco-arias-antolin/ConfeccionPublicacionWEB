Enlaces

La etiqueta que se usa para insertar enlaces es <a>

Los principales atributos de <a> son:

  • href: Especifica la URL. Puede contener una dirección de internet, un correo, un teléfono, una sección de la misma página o ejecutar javascript.
  • target: Cuyo valor puede ser: _blank (nueva pestaña), _parent (en el frame padre), _self (en el mismo frame) y _top (en todo el cuerpo de la ventana).

Para enlazar con la misma página hay que poner un id (por ejemplo "identificador") y hacerle referencia en href (#identificador).
Para que la página se desplace tiene que haber suficiente espacio por debajo.

Para cargar una página en un iframe con un enlace se hace así:

    <a href="https://www.direcciónDeLaPágina.com" target="iframe_1">Página.com</a>

    <iframe src="OtraPágina.html" name="iframe_1"></iframe>
