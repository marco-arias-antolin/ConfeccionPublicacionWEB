Identificación de los componentes que proporciona la herramienta Web.

Apartado 4.2.5.

Las barras de herramientas son fundamentales para la navegación por cualquier herramienta. Normalmente aparecen en la pantalla como una fila, columna o bloque que presenta botones o iconos permitiendo al usuario/a clicar sobre los botones para acceder a las distintas funciones.

Le solicitamos:

  • Describa las distintas partes que se pueden encontrar en la barra de herramientas de Visual Studio: ✓
    • Explorador: contendrá una visual directa sobre los archivos que está trabajando.
    • Buscar: buscar y reemplazar palabras en su archivo o carpetas. Puede buscar por palabras completas, hacer coincidir mayúsculas y/o minúsculas, buscar partes de palabras o expresiones regulares.
    • Control de código fuente: sirve para conectar a una carpeta compartida en Git para desarrollar código junto a otros programadores. Es una especie de carpeta compartida en tiempo real.
    • Ejecución y depuración: ejecuta y depura código.
    • Extensiones: una de las opciones más interesantes para buscar plugins para añadir a Visual Code y conseguir un flujo de trabajo más interesante y adaptado a sus necesidades.
    • Github: permite la integración con GitHub pudiendo crear y modificar repositorios remotos
  • Explique los pasos que debe seguir para obtener la ayuda de esta herramienta web (Versión: 1.77.0) ✓
    1. Introducción a VS Code
      • Ir al menú de Ayuda
      • Seleccionar Bienvenido
      • Click en Abrir un tutorial
      • Seleccionar "Introducción a VS Code" y completar sus tres partes
    2. Documentation for Visual Studio Code
      • Ir al menú de Ayuda
      • Seleccionar Documentación
      • Redirige a esta página: Documentation for Visual Studio Code
    3. keyboard-shortcuts-windows
      • Ir al menú de Ayuda
      • Seleccionar Referencia de métodos abreviados de teclado
      • Redirige a esta página: keyboard-shortcuts-windows.pdf
    4. Visual Studio Code Tips and Tricks
      • Ir al menú de Ayuda
      • Seleccionar Consejos y trucos
      • Redirige a esta página: Visual Studio Code Tips and Tricks
    5. Visual Studio Code Introductory Videos
      • Ir al menú de Ayuda
      • Seleccionar Tutoriales de vídeo
      • Redirige a esta página: Visual Studio Code Introductory Videos
