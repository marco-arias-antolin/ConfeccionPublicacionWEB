Los tipos de datos

Enunciado

En este foro le pedimos que reflexione sobre cuáles son los tipos de datos que se utilizan para representar y almacenar los valores de las variables en la elaboración de scripts.

Actividad asociada al CE1.3

Explicar los tipos de datos que se utilizan para representar y almacenar los valores de las variables en la elaboración de scripts, de acuerdo a las especificaciones técnicas del lenguaje.

Solución

En JavaScript, existen varios tipos de datos utilizados para representar y almacenar valores en variables. Estos tipos de datos incluyen:

  1. Números (Numbers): Se utilizan para representar valores numéricos, ya sean enteros o de punto flotante. Por ejemplo, 1, 3.14, -7, etc.
  2. Cadenas de texto (Strings): Se utilizan para representar secuencias de caracteres. Las cadenas de texto se encierran entre comillas simples ('') o comillas dobles (""). Por ejemplo, "Hola", 'JavaScript', "123", etc.
  3. Booleanos (Booleans): Se utilizan para representar valores de verdadero (true) o falso (false). Estos valores son útiles para realizar evaluaciones y tomar decisiones en el código. Por ejemplo, true, false.
  4. Arreglos (Arrays): Se utilizan para almacenar una colección ordenada de valores en una sola variable. Los elementos de un arreglo pueden ser de cualquier tipo de dato, incluyendo números, cadenas, objetos, etc. Los arreglos se declaran usando corchetes ([]). Por ejemplo, [1, 2, 3], ["manzana", "naranja", "plátano"].
  5. Objetos (Objects): Se utilizan para almacenar colecciones de pares clave-valor, donde cada valor está asociado a una clave única. Los objetos se declaran utilizando llaves ({}) y consisten en propiedades y métodos. Por ejemplo, {nombre: "Juan", edad: 25, ciudad: "Madrid"}.
  6. Valores nulos (null) y indefinidos (undefined): null se utiliza para representar la ausencia intencional de cualquier objeto o valor, mientras que undefined se utiliza para representar variables sin un valor asignado o propiedades no definidas en un objeto.

Estos son los tipos de datos básicos en JavaScript. Además de ellos, JavaScript también proporciona otros tipos de datos como funciones, fechas y expresiones regulares, que se utilizan para tareas más específicas.

En JavaScript, los tipos de datos que se utilizan para representar y almacenar valores en las variables, de acuerdo a las especificaciones técnicas del lenguaje, son los siguientes:

  1. Undefined: Representa una variable que no ha sido asignada o a la que no se le ha dado un valor. Es el valor predeterminado de una variable recién declarada. 
    let x;
console.log(x); // undefined
  2. Null: Representa la ausencia intencional de cualquier objeto o valor. Se utiliza para indicar que una variable no tiene un valor válido. 
    let y = null;
console.log(y); // null
  3. Boolean: Representa un valor lógico que puede ser verdadero (true) o falso (false). Se utiliza en lógica condicional y operaciones de control de flujo. 
    let isActive = true;
let isLoggedIn = false;
  4. Number: Representa valores numéricos, ya sean enteros o de punto flotante. También se utiliza para representar valores especiales como Infinity y NaN (Not a Number). 
    let a = 42; // número entero
let b = 3.14; // número de punto flotante
let c = Infinity; // infinito positivo
let d = -Infinity; // infinito negativo
let e = NaN; // valor inválido de número
  5. String: Representa una secuencia de caracteres. Se utiliza para almacenar texto y se puede encerrar entre comillas simples ('') o comillas dobles (""). 
    let greeting = "Hola";
let name = 'Juan';
  6. Symbol: Introducido en ECMAScript 6, representa un identificador único e inmutable. Se utiliza principalmente como claves de propiedades de objetos. 
    let id = Symbol("id");
let obj = { [id]: "12345" };
  7. Object: Representa un objeto, que es una colección de pares clave-valor. Las propiedades de un objeto pueden ser de cualquier tipo de datos, incluyendo otros objetos. 
    let person = {
  name: "Juan",
  age: 25,
  city: "Madrid"
};
console.log(person); // Object

Estos son los tipos de datos principales en JavaScript según las especificaciones técnicas del lenguaje. Sin embargo, JavaScript también es un lenguaje dinámicamente tipado, lo que significa que las variables pueden cambiar de tipo durante la ejecución del programa.
