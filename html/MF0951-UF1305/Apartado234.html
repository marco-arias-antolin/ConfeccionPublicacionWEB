Citar instrucciones para realizar operaciones de entrada y salida de datos

Enunciado

En esta nueva actividad seguimos practicando son los lenguajes de guion. Para ello, le solicitamos:

  • En primer lugar, programe un pequeño código dentro de un documento HTML capaz de recoger datos por pantalla y luego mostrarlos.
  • El programa deberá abrir una ventana y pedir nombre y edad.
  • Después deberá escribir Se llama “nombre que se haya introducido” y tiene “edad que se haya introducido” años.
  • Cite las instrucciones proporcionadas para realizar las operaciones de entrada y salida.
  • Compruebe el funcionamiento de la web en distintos navegadores

Actividad asociada al CE1.5

Citar las instrucciones proporcionadas por el lenguaje de guion para realizar operaciones de entrada y salida de datos, de acuerdo a las especificaciones técnicas del lenguaje.

Solución

Para solicitar datos de entrada al usuario, se puede usar la función prompt() que muestra una ventana emergente con un campo de entrada de texto. El usuario puede ingresar datos y hacer clic en "Aceptar" para confirmar la entrada. Por ejemplo:

var nombre = prompt("Introduce tu nombre:");

Para mostrar datos de salida al usuario, se puede utilizar la función alert() que muestra una ventana emergente con un mensaje. Por ejemplo:

var mensaje = "Hola, mundo!";
alert(mensaje);
También se puede usar la función console.log() para imprimir datos en la consola del navegador o del entorno de desarrollo. Esta función es útil para propósitos de depuración o para mostrar información en el registro de la consola. Por ejemplo: 
var nombre = "Juan";
var edad = 25;
console.log(`El nombre es ${nombre} y la edad es ${edad}`);
