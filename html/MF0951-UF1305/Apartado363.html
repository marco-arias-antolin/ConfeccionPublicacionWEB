Utilizar objetos del documento para permitir la interactividad entre el script y el usuario

Enunciado

La interactividad sirve de ayuda para escribir scripts flexibles, con una capacidad de aplicación más amplia. No se necesita especificar toda la información de entrada de antemano.

En la presente actividad, le solicitamos:

  • Cree un script en el que se le pida al usuario insertar su nombre y su año de nacimiento y, posteriormente, se muestre por pantalla con la siguiente frase: Hola, “nombre del usuario” tiene “años del usuario”.
  • Para sacar la edad del usuario se deberá usar una variable que coja el año actual, es decir, no vale utilizar var año=2021;.
  • Identifique los elementos de interactividad del script y los métodos usados en dicho script.

Actividad asociada al CE2.3

Describir e identificar los objetos del documento que permiten añadir interactividad entre el usuario y el script, así como sus propiedades y métodos.

Solución

  • Con prompt() y alert()
  • Con un formulario


Nota: El calculo puede ser inexacto si todavía no se han cumplido los años en el año actual. Para solucionarlo habría que introducir la fecha de nacimiento en vez de sólo el año.

Actividad asociada al CE2.3

Para añadir interactividad entre el usuario y el script en un documento HTML, existen varios objetos del documento (DOM) que se utilizan comúnmente. Aquí describo algunos de los objetos principales y sus propiedades y métodos relevantes:

  1. document: Este objeto representa todo el documento HTML cargado en el navegador. Proporciona acceso a otros objetos y métodos útiles para interactuar con el documento.
    Propiedades destacadas:
    • document.getElementById(id): Devuelve una referencia al elemento del documento con el ID especificado.
    • document.querySelector(selector): Devuelve el primer elemento que coincida con el selector CSS especificado.
    • document.querySelectorAll(selector): Devuelve una lista de todos los elementos que coinciden con el selector CSS especificado.
  2. element: Los elementos HTML son objetos que representan las etiquetas HTML en el documento. Pueden ser manipulados para añadir interactividad.
    Propiedades y métodos destacados:
    • element.textContent: Obtiene o establece el contenido de texto de un elemento.
    • element.value: Obtiene o establece el valor de un elemento de formulario.
    • element.addEventListener(evento, función): Registra un evento en el elemento y especifica la función que se ejecutará cuando ocurra el evento.
    • element.innerHTML: Obtiene o establece el contenido HTML del elemento.
  3. window: El objeto window representa la ventana del navegador y proporciona métodos y propiedades relacionados con la ventana actual.
    Propiedades y métodos destacados:
    • window.alert(mensaje): Muestra una ventana emergente con el mensaje especificado.
    • window.prompt(mensaje, valorPredeterminado): Muestra una ventana emergente que solicita al usuario que ingrese un valor.
    • window.confirm(mensaje): Muestra una ventana emergente con un mensaje y los botones de confirmación "Aceptar" y "Cancelar".

Estos son solo algunos ejemplos de objetos del documento que permiten añadir interactividad en JavaScript. El Document Object Model (DOM) proporciona una amplia gama de objetos, propiedades y métodos para interactuar con el contenido y la estructura de un documento HTML.
