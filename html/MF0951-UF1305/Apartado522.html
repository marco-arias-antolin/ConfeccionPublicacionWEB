Explicación de los objetos del navegador y uso de sus propiedades y métodos

Enunciado

Los navegadores proporcionan una serie de objetos organizados de acuerdo con una jerarquía a los que se puede acceder con JavaScript. Estos objetos se ajustan al denominado DOM (Document Object Model).

Para entender las posibilidades que ofrecen los objetos del navegador:

Cree un documento en el que, al pulsar un botón, muestre la siguiente información del navegador:

  • Nombre de la aplicación
  • Versión de la aplicación
  • Plugins soportados por la aplicación.

Actividad asociada al CE2.1

Explicar los objetos del navegador, así como sus propiedades y métodos, que se utilizan para añadir funcionalidad a las páginas web teniendo en cuenta las especificaciones técnicas del lenguaje.

Solución

Información del Navegador

Actividad asociada al CE2.1

El objeto navigator proporciona información sobre el navegador y el entorno en el que se está ejecutando el código JavaScript. Contiene propiedades y métodos que permiten acceder a detalles específicos del navegador. A continuación se presentan algunas de las propiedades y métodos más comunes del objeto navigator:

  1. navigator.userAgent: Propiedad que devuelve una cadena que representa el agente de usuario (User Agent) del navegador, que proporciona información sobre el navegador y el sistema operativo utilizado.
  2. navigator.appName: Propiedad que devuelve el nombre del navegador. Por ejemplo, "Netscape" o "Microsoft Internet Explorer".
  3. navigator.appVersion: Propiedad que devuelve la versión del navegador. Por ejemplo, "5.0 (Windows; en-US)".
  4. navigator.platform: Propiedad que devuelve el sistema operativo en el que se está ejecutando el navegador. Por ejemplo, "Win32" o "MacIntel".
  5. navigator.language: Propiedad que devuelve el lenguaje preferido del usuario, en formato de código de idioma. Por ejemplo, "es-ES" para español de España.
  6. navigator.plugins: Propiedad que devuelve una lista de los plugins instalados en el navegador. Cada plugin se representa como un objeto con propiedades como name (nombre), filename (nombre del archivo), description (descripción) y más.
  7. navigator.cookieEnabled: Propiedad booleana que indica si las cookies están habilitadas en el navegador.

Estas son solo algunas de las propiedades comunes del objeto navigator. También hay otros métodos y propiedades más específicos que pueden variar según el navegador y su implementación.
