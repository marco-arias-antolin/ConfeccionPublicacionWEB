Identificar eventos incluidos en el script

Enunciado

Dado el siguiente código que sirve para crear una lista de la compra, se deberán crear determinados scripts con eventos que permitan resetear la lista cuando el usuario desee, la lista no debe añadir nuevas líneas cuando el usuario no haya escrito nada.

<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta name="description" content="Creación de nodos mediante javascript">
  <title>Ejercicio - La lista de la compra</title>
</head>
<body>
  <form action="#">
    <input type="text" class="introduccionDatos" id="introduccionDatos">
    <input type="button" value="Añadir a la lista" id="anadir">
    <input type="button" value="Reiniciar" id="reiniciar">
  </form>
  <ul id="elementoUl">
  </ul>
  <script>
    window.onload = init;
    function init() {
      document.getElementById("anadir").addEventListener("click", anadirElemento);
      document.getElementById("reiniciar").addEventListener("click", reiniciarLista);
    }
    function anadirElemento() {
      //...
    }
    function reiniciarLista() {
      //...
    }
  </script>
</body>
</html>

Actividad asociada al CE3.3

Identificar los eventos incluidos en el script para distinguir las interacciones con el usuario.

Solución

El código de las funciones es el siguiente:

function anadirElemento() {
  var input = document.getElementById("introduccionDatos");
  var texto = input.value.trim();

  if (texto !== "") {
    var lista = document.getElementById("elementoUl");
    var nuevoElemento = document.createElement("li");
    nuevoElemento.textContent = texto;
    lista.appendChild(nuevoElemento);
    input.value = "";
  }
}

function reiniciarLista() {
  var lista = document.getElementById("elementoUl");
  lista.innerHTML = "";
}

El ejercicio funcionando:

Actividad asociada al CE3.3

En el script proporcionado, se incluyen dos eventos que permiten interactuar con el usuario:

Evento "click" en el botón "Añadir a la lista":

document.getElementById("anadir").addEventListener("click", anadirElemento);

Este evento se dispara cuando el usuario hace clic en el botón "Añadir a la lista". El evento está asociado a la función "anadirElemento()", que se ejecutará cuando ocurra el evento.

Evento "click" en el botón "Reiniciar":

document.getElementById("reiniciar").addEventListener("click", reiniciarLista);

Este evento se activa cuando el usuario hace clic en el botón "Reiniciar". El evento está asociado a la función "reiniciarLista()", que se ejecutará cuando ocurra el evento.

Estos eventos permiten al usuario interactuar con la página web al hacer clic en los botones correspondientes. El evento "click" se desencadena cuando se hace clic en un elemento específico, lo que permite ejecutar las funciones asociadas para realizar acciones como añadir elementos a la lista o reiniciarla.
