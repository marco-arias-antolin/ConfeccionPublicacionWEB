Alta de páginas web

Enunciado

  1. Actualice su sitio web con una nueva página y compruebe la visibilidad de la misma con un navegador.
  2. Compruebe con varios buscadores web que su página web es visible en internet e indique qué acciones puede realizar en la web para que el buscador la encuentre de forma más rápida.
  3. A continuación, se le pide que instale el navegador brave e instale un plugin para cambiar el tema del navegador. Este plugin se denomina Stargazers.

Actividad asociada al CE4.3

Dar de alta las páginas publicadas en buscadores para exponer las informaciones contenidas en ellas:

  1. Seleccionar los buscadores.
  2. Incluir los descriptores en las páginas realizadas utilizando las etiquetas apropiadas.
  3. Usar una aplicación de publicación automática para publicar las páginas realizadas.
  4. Publicar manualmente las páginas que no puedan publicarse con un procedimiento automático.
  5. Verificar periódicamente la funcionalidad de la página publicada.
